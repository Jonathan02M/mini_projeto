## 📌 Descrição do Projeto

O **Controle de Hábitos** é um aplicativo desenvolvido em Flutter com o objetivo de auxiliar usuários a acompanharem e manterem hábitos diários, como estudar, beber água ou praticar exercícios.

O app permite marcar hábitos como concluídos, acompanhar o progresso diário e visualizar estatísticas de desempenho em um layout moderno inspirado em aplicativos fitness.

---

## 🚀 Funcionalidades

✔ Listagem de hábitos
✔ Marcar hábitos como concluídos (checkbox)
✔ Adicionar novos hábitos
✔ Remover hábitos
✔ Persistência de dados (SharedPreferences)
✔ Sistema de pontuação (streak 🔥)
✔ Barra de progresso diária
✔ Tela de estatísticas (estilo fitness)
✔ Navegação entre telas
✔ Interface moderna com tema dark

---
